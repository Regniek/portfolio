Portafolio

Este portafolio comenzo a realizarce el 22/06/2020, inicialmente estaba pensado para ser una SPA, pero luego poco a poco se fue transformando, esta hecho con diferentes herramientas de desarrollo, entre ellas:

HTML


el HTML fu implmentado gracias al preprosesador PUG el cual permito generar el Header y Footer mas flexibles y poder hacer un cambio de paginas mas armonico y en poco tiempo.



CSS con SASS




Utilizamos SASS un preprocesador de CSS para poder generar una pagina mas comoda al momento de generar un cambio en su estilo, ya que este preprocesador nos permitio agregar variables y funciones de de tal manera que nos permite estandarizar su estilo y genera una ayuda al momento de aplicarle mejora continua a la misma.


JavaScript
